== Description ==
Plugin Interface for the [Toybox](https://www.toyboxsystems.com/) Website Feedback Tool. Allow your customers to give feedback on your website easily.

We would love to hear your feedback in a [review](https://wordpress.org/support/plugin/wk-toybox/reviews/). It helps us improve and expand the plugin according to your needs.
If you have any questions or feature requests, feel free to contact us via support@webkinder.ch.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wk-toybox directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Toybox screen to configure the plugin
